# Co-Editor 问题修复与优化任务

## 需要修复的问题：

### Bug 1: 公开文档点击进入没有呈现编辑页面
- 问题：点击编辑后，编辑器容器没有显示
- 位置：editor.js 中的 showEditor() 函数

### Bug 2: 创建文档的弹窗没有居中
- 问题：创建文档模态框位置不居中
- 位置：style.css 中的 .modal 样式

### Bug 3: 密码弹窗在蒙版下一层，无法点击和输入，且没有居中显示
- 问题：z-index 顺序错误，导致模态框被蒙层遮挡
- 位置：style.css 中的 .modal 和 .password-overlay 的 z-index

## 需要优化的功能：

### 优化 1: 创建相同名字文档的提示
- 在创建前检查文档ID是否已存在
- 如果已存在，提示用户"文档ID已被占用"

### 优化 2: 文档名字限制（英文数字下划线横线）
- 使用正则表达式验证：`^[a-zA-Z0-9_-]+$`
- 只允许：a-z, A-Z, 0-9, _, -
- 长度限制：3-50个字符

### 优化 3: 密码设置的UI效果
- 优化密码输入框的样式
- 添加密码显示/隐藏切换按钮
- 改进密码复选框样式

### 优化 4: 字符长度限制
- 文档ID：3-50个字符
- 密码：4-100个字符（可选）

## 技术要求：
- 使用 Claude 3.5 Sonnet 模型
- 保持原有架构不变
- 确保修复后的代码质量
- 添加必要的错误处理和用户提示
